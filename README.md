# Arduino GRBL CNC Plotter

An open-source CNC plotter built using **Arduino Uno**, **GRBL firmware**, and **CNC Shield V3**.  
This project demonstrates reliable 2D motion control using G-code streamed via **Universal G-code Sender (UGS)**.
